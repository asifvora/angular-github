# Angular Github
Angular Github Viewer using angular7.


## Getting Started

For help getting started with Angular, view online
[documentation](https://angular.io/).

#### 1. [Setup Angular](https://github.com/angular/angular-cli)

#### 2. Clone the repo

```sh
$ https://github.com/asifvora/angular-github.git
$ cd angular-github/
```

#### 3. Install dependency

```sh
$ npm install
```

#### 4. Run the app

```sh
$ ng serve
```
#### 5. Production build

```sh
$ ng build
```
